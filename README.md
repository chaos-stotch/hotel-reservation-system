# Hotel Reservation System

## O Hotel Reservation System é uma plataforma de reservas e gestão de quartos para hotelaria.

O escopo do projeto é delimitado pelos seguintes pontos:

1. reservas de quartos;
    1. agendamento
    2. cancelamento
2. gestão de quartos;
    1. validações de regra de negócio automáticas
    2. visualização da situação dos quartos para administrador autenticado do sistema

Entidades principais, atributos e regras de negócio estão documentadas [aqui](docs/domain.md). Exemplificação do esquema arquitetural se encontra [aqui](docs/architecture.md).
